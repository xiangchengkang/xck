# Visual Studio Code

## 简介

VS Code 是轻量级代码编辑器，拥有丰富插件生态。

## 下载入口

- 官方下载：https://code.visualstudio.com/Download

## 推荐插件

- Chinese (Simplified)
- ESLint
- Prettier
- GitLens
- Markdown All in One

## 使用教程

### 基础配置

1. 安装中文语言包
2. 打开设置，启用自动保存
3. 配置格式化工具（Prettier）

### 工作区建议

- 为每个项目保存独立工作区配置
- 使用 `settings.json` 统一团队代码风格
