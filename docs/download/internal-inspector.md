# 内网巡检助手

## 软件简介

内网巡检助手用于自动检测服务器基础运行状态，支持巡检报告导出和异常提醒。

## 使用教程

### 1. 安装

1. 下载 zip 压缩包并解压
2. 双击 `internal-inspector.exe`
3. 首次启动按提示选择巡检目标

### 2. 基础使用

1. 新建巡检任务
2. 选择主机清单
3. 点击“开始巡检”
4. 导出报告（HTML / Markdown）
