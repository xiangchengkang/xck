# Git

## 简介

Git 是分布式版本控制工具，用于代码管理与协作开发。

## 下载入口

- 官方下载：https://git-scm.com/downloads

## 常用版本建议

- Windows：安装最新版稳定版
- Linux：建议使用系统包管理器安装
- macOS：建议通过 Homebrew 安装

## 使用教程

### 快速配置

```bash
git config --global user.name "your-name"
git config --global user.email "you@example.com"
```

### 常用命令

```bash
git clone <repo-url>
git checkout -b feature/xxx
git add .
git commit -m "feat: ..."
git push
```
